While this chapter features no results from experiments, it still performs some preliminary analysis on both the PATH dataset and the real-world dataset.
The corresponding scripts are all located in the `datasets` folder.

## PATH Dataset
For the PATH dataset, the Fourier, downsampling and autocorrelation analyses are done with the `path_fourier.py`, the `path_downsampling.py` and the `path_autocorrelation.py`, respectively.

The code to generate the illustrative sine wave plot is provided in `sine_example.py`.

## Real-world Dataset
Unfortunately, this proprietary dataset is subject to strict confidentiality levels and therefore cannot be published.
The Fourier, downsampling and autocorrelation analyses are performed with the `prop_fourier.py`, the `prop_downsampling.py` and the `prop_autocorrelation.py` scripts, respectively, and are almost identical to the ones for the PATH dataset.
